**Операции над множествами:**
Объединением двух множеств X и Y называется множество, обозначаемое X∪Y и состоящее из элементов, принадлежащих хотя бы одному из множеств X или Y.
         Пересечением множеств X и Y называется множество, обозначаемое X∩Y состоящее из элементов, принадлежащих каждому из множеств X и Y.
         Разностью множеств X и Y называется множество, обозначаемое X\Y и состоящее из всех элементов X, не принадлежащих Y.
        Симметричной разностью множеств X и Y называется множество: X∆Y = (X\Y)∪(Y\X).
        Дополнением к множеству X относительно универсального множества U называется множество X' = U\X.

8) Пересечение (A ∩ B):
1. Создается копия множества A (С)
2. Для каждого элемента множества A проверяется его наличие в В
3. Элементы, найденные в обоих множествах, добавляются в результат множества С
4. Просматриваем множество С и удаляем повторы
5. Выводим информацию пользователю на экран множество С

8) Разность (A \ B):
1. Создается копия множества A (С)
2. Для каждого элемента A проверяется его отсутствие в B
3. Элементы, уникальные для A, составляют результат и добавляются в множество С
4. Просматриваем множество С и удаляем повторы
5. Выводим информацию пользователю на экран множество С

9) Разность (B \ A):

1. Аналогично предыдущей операции, но для элементов B относительно A

10) Симметричная разность (A Δ B):
1. Создадим пустое множество C для хранения результата.
2. Для каждого элемента из множества A: Проверить, содержится ли этот элемент в множестве B. Если элемента нет в B, добавить его в C.
3. Для каждого элемента из множества B: Проверить, содержится ли этот элемент в множестве A. Если элемента нет в A, добавить его в C.
4. Выводим информацию пользователю на экран множество С

11) Дополнение множеств:
1. Создается копия множества A (С)

4. Выводим информацию пользователю на экран множество С
