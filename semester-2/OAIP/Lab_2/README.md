## Цель работы
Изучить алгоритмы работы с динамическими структурами данных в виде стека.

Написать программу по созданию, добавлению, просмотру и решению приведенных далее задач для однонаправленного линейного списка типа Stack. Реализовать сортировку стека методами, рассмотренными в подразделе 3.1.  

### Задание 5
Из созданного списка удалить элементы, заканчивающиеся на цифру 5.

### Дополнительное задание 
В созданном списке поменять местами элементы, содержащие максимальное и минимальное значения.
